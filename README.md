# Cat Bond Portfolio Diversification Analyzer

## Overview
This project analyzes a portfolio of catastrophe bonds and evaluates diversification across regions and disaster types.

## Features
- Portfolio investment analysis
- Expected loss calculation
- Risk-adjusted return calculation
- Region-wise risk analysis
- Diversification scoring
- Risk ranking of catastrophe bonds
- Data visualization using Matplotlib

## Technologies Used
- Python
- Pandas
- Matplotlib

## Project Structure

├── main.py
├── portfolio_data.csv
├── requirements.txt
└── README.md
